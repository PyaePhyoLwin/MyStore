Security & Download Verification

Every release includes SHA256 values to help verify file integrity.

How to verify checksums

  1. Download an app file from this website.
  2. Copy the SHA256 value shown on the app detail page.
  3. Run one of these commands and compare output exactly:
# macOS / Linux
shasum -a 256 file-name.ext

# Windows PowerShell
Get-FileHash .\file-name.ext -Algorithm SHA256

Code signing

Official installers are signed where platform signing is supported (Android, macOS, Windows). Always check publisher identity before installing.

Update policy

Critical fixes are prioritized and announced in release notes. Older versions stay available for rollback until end-of-support.
